# AmroAssessment

## Key Features

- List trending movies
- Sort movies by popularity, title, or release date
- Filter movies by genre
- Access movie details
- Navigate to IMDb external link
- Dark mode support

## Design Patterns & Choices

- MVVM + Clean Architecture
- Multi-Module Architecture
- Dependency Injection
- Repository pattern
- Mapper pattern
- Offline first

## Tech Stack

- UI - Jetpack Compose
- Navigation - Navigation 3
- Dependency Injection - Hilt
- Asynchronous work - Coroutines
- Network - Retrofit
- Local database - Room
- Pagination - Paging 3
- Image loading - Coil 3
